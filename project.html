STUDENT ATTENDANCE MONITORING SYSTEM


Development of Automated Student Attendance Monitoring System using Image Processing, is a prototype that automatically records attendance upon registration of the student to the system. It aims to minimize the time execution and to outsmart the traditional attendance system like roll-calls, filling signatures, and timeclocks. The system lessens the time execution in acquiring attendance record because it records studentsâ€™ information and their faces patterns, which serves as verification data to generate an attendance report.


The system was developed using Python programming language for the construction of codes, PyQt5 designer for the graphical user interface and Numpy and OpenCV for libraries, with Local Binary Pattern Histogram algorithm for the face recognition prediction and SQLite for the database management.
